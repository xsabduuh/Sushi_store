# Sushi Store

## Project Overview
Sushi Store is a web application that allows users to browse and order a variety of sushi dishes. The application serves both customers looking to enjoy sushi at home and restaurant owners who want to manage their online sushi menu efficiently.

## Features
- **User Authentication:** Secure login and registration process to ensure user data privacy.
- **Menu Display:** Browse a comprehensive list of sushi options, including images, descriptions, and prices.
- **Order Management:** Users can add sushi to their cart, view cart details, and checkout seamlessly.
- **Admin Dashboard:** Restaurant owners can add, update, and remove sushi items from the menu.
- **Responsive Design:** The application is optimized for mobile, tablet, and desktop views.

## How to Use
### For Users:
1. **Sign Up:** Create an account by providing your email and password.
2. **Login:** Access your account using your credentials.
3. **Browse Menu:** Navigate through the sushi menu and select your favorite dishes.
4. **Place Order:** Add items to your cart and proceed to checkout.

### For Restaurant Owners:
1. **Admin Login:** Sign in to your admin account.
2. **Manage Menu:** Use the dashboard to add, update, or remove sushi items.
3. **View Orders:** Monitor customer orders in real-time.

## Installation
To set up the project locally, follow these steps:
1. Clone the repository: `git clone https://github.com/xsabduuh/Sushi_store.git`
2. Navigate to the project directory: `cd Sushi_store`
3. Install dependencies: `npm install`
4. Start the application: `npm start`

## Technologies Used
- **Frontend:** React.js, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
